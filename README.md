# BOOK & GO - Movie Ticket Booking System

## Project Overview
This is a React.js based Movie Ticket Booking System developed for the BCA Semester-5 Software Development Project (SDP-1).

## Tech Stack
- **Frontend Library**: React.js (Vite)
- **Routing**: React Router DOM
- **Styling**: Bootstrap 5 & Custom CSS
- **HTTP Client**: Axios (for API integration)

## Folder Structure
- `src/assets`: Static assets like images and global styles.
- `src/components`: Reusable UI components (Navbar, Footer, etc.).
- `src/pages`: Main application views (Login, MovieList, Booking, Admin).
- `src/services`: API configuration and mock data.
- `src/context`: Context providers for state management (e.g., Auth).
- `src/utils`: Helper functions.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Features
- **User Module**: Browse movies, view details, select seats, book tickets.
- **Admin Module**: Manage movies and view bookings.
- **Authentication**: Login/Register (Mock implementation).

## Academic Note
This project is designed as an MVP (Minimum Viable Product) to demonstrate core React concepts including Components, Props, State, Hooks (useState, useEffect), and Routing.
